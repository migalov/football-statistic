## Установка на локальном серверве
- Установить [NodeJS](https://nodejs.org/)
- Склонировать репозиторий
- Открыть командную строку и ввести:

```sh
cd football-statistic
yarn
```

## Разработка
Запуск режима разработки:
```sh
yarn start
```
